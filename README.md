# Napster Mother's Day 2026 — Landing Page Preview

Team preview of the "Make Your Favorite Mom a Song" landing page for Mother's Day 2026.

**Campaign window:** May 1–11, 2026
**Winner announced:** May 14, 2026
**Prize:** $500 sweepstakes for a public social post tagged `#MadeOnNapster`
**Final build target:** Webflow. This single-file HTML is a working mockup for review + dev handoff.

## Enabling GitHub Pages

1. Settings → Pages
2. Source: **Deploy from a branch**
3. Branch: `main` / `/ (root)`
4. Save. Live URL appears in ~1 minute.

## Notes for reviewers

- `<meta name="robots" content="noindex, nofollow">` is set so this preview won't be indexed by search engines while the campaign is pre-launch.
- The "Create her song" CTA currently redirects to `https://app.napster.com/create?companion_id={id}&prompt={urlencoded}` — the live endpoint. Please flag any redirect issues.
- Portrait crops, genre tags, and sample prompts are still open to feedback.
